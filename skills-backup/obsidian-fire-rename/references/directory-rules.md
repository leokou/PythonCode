# 目录展示规则参考

## 格式规则

1. 第一行：文件标题，用 `#` 开头
2. 分类：直接写文字，不要加 `#`、`-`、`---` 等任何符号
3. 文件：用 tab 缩进 + `[[文件名]]` 链接
4. 分类之间：空一行
5. 每个分类下至少有1个文件，空分类不要写

## 分类维度

### 按重要程度（推荐优先使用）

系统文件 → 基础材料 → 核心功能 → 工具配置 → 参考资料 → 其他

### 按流程阶段

- 项目类：启动 → 执行 → 收尾 → 复盘
- 案件类：基础材料 → 一审 → 二审 → 再审
- 学习类：入门 → 进阶 → 高级 → 精通
- 开发类：需求 → 设计 → 开发 → 测试 → 部署

### 按内容类型

教程 | 配置 | 问题解决 | 技巧 | 对比 | 笔记 | 模板 | 案例

## 完整模板

```
# 🧩 目录-xxx

分类名称1
	[[文件1]]
	[[文件2]]

分类名称2
	[[文件3]]
	[[文件4]]
```

## 常见错误

1. ❌ 分类前加 `#`、`-`、`---` 等符号
2. ❌ 文件前加 `-` 符号
3. ❌ 空分类没有文件
4. ❌ 分类之间没有空行
5. ❌ 文件没有用 `[[]]` 链接格式
6. ❌ 文件没有用 tab 缩进

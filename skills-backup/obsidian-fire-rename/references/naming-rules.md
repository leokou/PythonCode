# 命名规则参考

## 完整规则

详见 `D:\Obsidian\LeoDiary\script\🥝Obsidian 自动生成标题 -命名规则.md`

## 快速参考

### 标准格式

```
主题 - 类型 @ 说明
```

### 字段说明

| 字段 | 定义 | 规则 |
|------|------|------|
| 主题 | 文件最核心对象 | 使用明确实体，不加情绪词 |
| 类型 | AI分类依据 | 使用固定词库，不要创造 |
| 说明 | 文件保存价值 | 不重复标题，回答"有什么价值" |

### 类型词库

**知识学习类**：概念、原理、笔记、教程、指南、手册、总结、对比、清单、案例

**工作项目类**：需求文档、设计方案、技术方案、实施方案、会议记录、分析报告、复盘总结

**软件工具类**：安装、配置、教程、指南、对比、测评、速查

**状态（仅项目文件）**：草稿、初稿、V1、V2、修订版、测试版、定稿、正式版、归档

### @说明类型

| 类型 | 格式 | 示例 |
|------|------|------|
| 用途说明 | `@ 用于xxx` | `@ 用于服务器部署` |
| 内容说明 | `@ 包含xxx` | `@ 语法、案例、最佳实践` |
| 场景说明 | `@ xxx场景使用` | `@ 手机网络分流场景` |
| 关联说明 | `@ 关联xxx` | `@ LEO OS智能调用体系` |

### 特殊符号

| 符号 | 作用 | 示例 |
|------|------|------|
| `-` | 一级层级分隔 | `Python - 教程` |
| `@` | 补充上下文 | `Python - 教程 @ 快速入门` |
| `&` | 多个主体关联 | `Claude Code & Cursor - 对比` |
| `+` | 组合升级 | `Vue3 + Hono - 架构方案` |

### 判断流程

```
文件回答"是什么" → 主题 - 概念
文件回答"怎么做" → 主题 - 教程
文件回答"为什么" → 主题 - 原理分析
文件回答"怎么选择" → 主题 - 对比分析
```
